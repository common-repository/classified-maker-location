	Location for Classified Maker

== Description ==


Add location for Classified Maker ads.


### Classified Maker by http://pickplugins.com
* [Live Demo!&raquo;](http://www.pickplugins.com/demo/classified-maker/location/new-york/)
* [Plugin details!&raquo;](http://www.pickplugins.com/item/classified-maker)



<strong>Plugin Features</strong>

* Ads Count by Location
* Ads in Map
* Location submit form

<strong>Ads Count by Location</strong>

 `[classified_maker_ads_count_lcoaiton]`
 

<strong>Ads in Map</strong>

 `[classified_maker_ads_map]`

<strong>Location submit form</strong>

 `[classified_maker_new_location]`

<strong>Translation</strong>

Pluign is transaltion ready , please find the 'en.po' for default transaltion file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/


== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Classified Maker - Location</strong>" activate it.<br />
